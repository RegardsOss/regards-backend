
components/ : Contient les composants react de l'application
containers/ : Contient les wrapper de composants react. Il s'agit des controlers.
modules/ : Contient les modules de l'application.
          Un module est une partie de l'application contenant lui même
          des components et containers.
AdminApp.js/ : Entré de l'application.
index.js/ : fichier principal de routage de l'application pour react router
