Ce package contient les opérations permettant la compression de fichiers.

L'utilisation du design pattern stratégie permet d'offrir des possibilités d'extension. Dans la version initiale du SIPAD-NG, la compression ZIP, TAR et GZIP.

Vue statique

La classe CompressionFacade est la facade du paquetage, elle utilise le pattern Strategie pour masquer l'implementation concrete des compressions. Le contexte instancie les classes concretes à partir de l'enuméré CompressionTypeEnum qui lui est transmis. Puis il manipule la compression concrète par l'interface ICompression.
